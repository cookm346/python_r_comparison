# Python/R conversion cheatsheet: How to do everything you do in Python in R and vice verse

### Variable assignment

#### R
```r
x <- 2    # x = 2 also works though it is bad form
```

#### Python 
```python
x = 2
```
<br>

### Convert text to lowercase

#### R
```r
s <- "THIS is a STRING"
tolower(s)
```

#### Python 
```python
s = "THIS is a STRING"
s.lower()
```
<br>

### Convert text to uppercase

#### R
```r
s <- "THIS is a STRING"
toupper(s)
```

#### Python 
```python
s = "THIS is a STRING"
s.upper()
```
